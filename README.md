## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.


Check out demo site [Next.js](https://creative-starship-6bf2b1.netlify.app/) for more details.
